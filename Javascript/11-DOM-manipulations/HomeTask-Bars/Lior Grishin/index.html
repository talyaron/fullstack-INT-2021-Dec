Bar Charts

A bar chart or bar graph is a chart or graph that presents categorical data with rectangular bars with heights or lengths proportional to the values that they represent. The bars can be plotted vertically or horizontally. A vertical bar chart is sometimes called a column chart.

How to use?

Right click anywhere and select Inspect
In the console tab call function "createGraphBars(barArray)" - default value
To remove existed bars call function "removeBars( )"
Play yourself with the "Bar Chart" by calling the function and inserting your type array
Example: createGraphBars([25, 10, 65, 80, 45, 20, 71])
Array must be with numbers between 0 to 100.
